# GasoLink - Painel Administrativo

Este é o sistema de gerenciamento de versões e inteligência em combustíveis do ecossistema GasoLink.

## Acesso Restrito

O acesso a este painel é estritamente limitado a usuários autorizados.

- **Autenticação:** É necessário possuir um e-mail e senha previamente cadastrados no banco de dados oficial.
- **Segurança:** Tentativas de acesso sem credenciais válidas serão bloqueadas pela API.

## Tecnologias

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js / Google Cloud Run
- **Banco de Dados:** Firebase Firestore
